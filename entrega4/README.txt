Para utilizar las consultas tienes que asegurarte que
todos los valores que tienen un $ correspondan
a las variables que representan en php

Revisa el archivo conexionMongo.php para ver como conectarte a la base de datos de mongo
(necesario en todas las páginas que usen Mongo)
